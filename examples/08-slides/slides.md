# PageFire 幻灯片示例

`deploy_markdown` + `mode="slide"`

---

## 如何使用

```
deploy_markdown({
  markdown: "你的 Markdown 内容",
  mode: "slide",
  theme: "dark"
})
```

---

## 功能特性

- ← → 键盘翻页
- 代码自动高亮
- 三种主题：light / dark / sepia
- 触屏滑动支持
- 按 `P` 演示者模式
- 按 `O` 页面总览

---
