MCP 工具
11 个工具，覆盖所有场景
deploy_page
发布单个 HTML 字符串，最快路径，适合 AI 生成的落地页和报告。
deploy_markdown
Markdown → 精致排版网页，支持 light / dark / sepia 三种主题，代码高亮、表格、任务列表齐全。
deploy_docs
多篇 Markdown → 带左侧导航的文档站，必须含 index.md，.md 链接自动改写为 .html。
deploy_files
逐文件发布多页站点（HTML + CSS + JS），支持子目录结构，适合手写多页网站。
deploy_zip
上传 ZIP 打包产物，自动解包，支持 SPA 客户端路由模式（spa: true）。
set_access
随时修改访问控制：公开 / 密码保护 / 私有，无需重新发布内容。
list_deployments
查看当前 space 下所有部署，含 URL、状态、文件数、大小、创建时间。
get_deployment
查看单个部署的详细信息。
delete_deployment
删除指定部署，释放存储配额。
